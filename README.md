# Flip
Flip is web multiplayer platformer game.

### Prerequisites
MongoDB database 

### Installing
In order to install all depencies run:
```npm install 
```
To start database run:
```npm run db 
```

###  Developer mode
To run game in developer mode run:
```
npm start
```
###  Build and run for production
Build the project with:
```
npm run build 
```
start game: 
```
npm run app 
```
